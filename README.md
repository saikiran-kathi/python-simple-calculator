# Simple Calculator (Python)
## Description
This is a beginner-friendly Python project that performs basic arithmetic operations such as:
- Addition
- Subtraction
- Multiplication
- Division

The program runs in the command prompt and takes user input.

## Concepts Used
- input() function
- if-elif-else conditions
- Arithmetic operators
- Basic Python syntax

## How to Run
1. Open Command Prompt
2. Navigate to the project folder
3. Run the command:
   py main.py
